# markitdown-mcp

Converts documents (PDF, DOCX, XLSX, PPTX, HTML, TXT, MD) to Markdown.

## Tools

- **process_document**: Convert a file to Markdown. Returns `file_id` and `markdown` content.
- **get_markdown**: Retrieve stored markdown by `file_id`.
- **list_documents**: List all processed documents.
- **search_summaries**: Keyword search across stored markdown content.
- **get_summary**: Get metadata for a processed document.
